# GradQuest 데모데이 평가기준 대응표

## 대주제 대응

대주제: 흩어진 정보를, 쓸 수 있는 서비스로!

GradQuest는 성균관대학교 학부생이 GLS, 챌린지스퀘어, 학과 공지, 공식 학사 안내에 흩어진 졸업요건 정보를 하나의 개인화 진단 서비스로 연결합니다. 정보의 출처는 문서, 이미지, PDF, 웹 공지, 사용자 입력을 모두 포함하며, 결과는 단순 요약이 아니라 부족 요건, 인정내역, 다음 행동, 추천 과목, 3품 활동, AI 상담으로 이어집니다.

## 1. Service Differentiation

### 범용 LLM 대비 서비스 차별성

범용 챗봇은 사용자가 자신의 학번, 전공, 복수전공, 이수학점, 3품 현황, 졸업평가 조건을 직접 설명해야 합니다. GradQuest는 이 정보를 프로필과 이수내역으로 구조화해 개인별 졸업요건을 자동 계산합니다.

차별점:
- 학번·전공·복수전공·융합트랙·조기졸업 여부 반영
- GLS/챌린지스퀘어 이수내역을 요건별 인정내역으로 연결
- 부족 학점과 다음 학기 추천 과목 제시
- 3품 미충족 영역에 맞춘 비교과 추천
- 공식 출처 링크와 최종 확인 안내 포함

### 문제 정의와 타깃

타깃은 성균관대학교 학부생, 특히 복수전공·융합트랙·조기졸업처럼 조건이 복잡한 학생입니다. 이들은 “내가 졸업 가능한가”보다 “무엇이 부족하고 다음에 무엇을 해야 하는가”를 알고 싶어합니다.

## 2. Data Architecture & Process

정보 처리 파이프라인:

1. 정보 소스 수집
   GLS 성적표, 챌린지스퀘어 이수내역, 학과 졸업요건, SKKU 공식 공지

2. 문서/이미지 인식
   Upstage Document AI로 과목명, 학점, 성적, 수강학기, 비교과 프로그램, 인증영역을 추출

3. 구조화
   학생 프로필, 졸업요건, 교과목, 비교과, 졸업평가 체크리스트 스키마로 변환

4. 규칙 매칭
   입학연도, 전공형태, DS 예외, 3품, 졸업평가, 조기졸업 기준과 대조

5. 서비스 결과
   대시보드, 요건별 인정내역, 추천 과목, 비교과 추천, AI 상담으로 연결

## 3. Solution Depth

GradQuest는 단순 문서 요약이 아니라 판정 로직을 포함합니다.

- 총 졸업학점과 전공/교양 세부 부족분 계산
- 소프트웨어학과 DS 지정과목 예외 처리
- 3품 인증 영역별 완료 여부 확인
- 졸업평가 체크리스트 반영
- 조기졸업 GPA, 학기, 학점, 전체 요건 진행률 계산
- 각 졸업요건에 어떤 과목과 비교과가 연결되는지 확인

## 4. Service Impact

실사용성:
- 학생은 여러 사이트를 돌아다니지 않고 자신의 부족 조건을 한 화면에서 확인
- 다음 학기에 어떤 영역을 채워야 하는지 즉시 판단 가능
- 공식 공지 링크로 최종 확인 가능

확장성:
- 현재는 주요 페르소나와 자연과학캠퍼스/소프트웨어학과 중심
- 이후 학과별 졸업요건, 비교과, 장학금, 취업 프로그램 추천으로 확장 가능

## 5. Effective Use of Upstage

사용 제품:
- Upstage Document AI: GLS/챌린지스퀘어 이미지 또는 PDF에서 이수내역 추출
- Upstage Solar: 학생 프로필과 졸업요건 데이터를 기반으로 학사 Q&A 답변

기여도:
- 문서 인식을 통해 사용자의 수동 입력 부담을 줄임
- Solar가 단순 대화가 아니라 구조화된 학생 데이터와 규칙을 기반으로 답변
- Upstage API Key는 Vercel Serverless Function에서 관리해 프론트에 노출하지 않음

## 6. Presentation & Documentation

발표에서 강조할 문장:

“GradQuest는 그냥 졸업요건을 검색하는 서비스가 아니라, 흩어진 학사 문서와 학생의 실제 이수내역을 연결해 지금 부족한 조건과 다음 행동을 알려주는 성균관대 학부생 맞춤 졸업 내비게이터입니다.”

데모 순서:

1. 대시보드에서 전체 진행률과 부족 학점 확인
2. 요건별 인정내역에서 어떤 과목이 어떤 졸업요건을 채웠는지 확인
3. 3품 인증에서 부족 영역과 추천 비교과 확인
4. 조기졸업 페이지에서 GPA/학점/전체 요건 부족분 확인
5. AI 학사 도우미에서 “다음 학기에 무엇부터 채워야 해?” 질문
6. 문서 등록 페이지에서 Upstage Document AI 연결 구조 설명

## 심사위원 질문 대비

### 그냥 ChatGPT에 물어보면 되지 않나요?

ChatGPT는 사용자가 모든 조건과 이수내역을 직접 제공해야 합니다. GradQuest는 학교별 규칙, 학생 프로필, GLS/챌린지스퀘어 이수내역을 구조화해 개인별 판정과 다음 행동까지 연결합니다.

### 실제로 쓸 수 있나요?

MVP는 샘플 프로필과 문서 인식 흐름을 중심으로 구현했습니다. 실제 운영 시에는 학교 공식 데이터 연동과 학과별 규칙 검수 절차를 추가하면 학부생용 졸업 준비 도구로 확장할 수 있습니다.

### Upstage는 어디에 쓰이나요?

Document AI는 이미지/PDF에서 이수내역을 추출하고, Solar는 구조화된 졸업요건과 학생 프로필을 기반으로 Q&A를 제공합니다.
